# KickStartGolang
Golang submission template for Kick Start.

## Test cases input format
### Line 1:
- T => num. of test cases

### Line (2xT):
- n p => num. of player's rating, and, num. of players to pick

### Line (2xT) + 1:
- R(1) R(2) ... R(n) => each player's rating

## Example
- 2
- 5 2
- 1 3 5 7 8
- 5 2
- 2 4 6 7 9
